How every derived value is built

calcRest daysGame ›
Where it's from

Not scraped — read straight off the schedule already in games.

Sources

SofaScore team schedule → games.date.

How it's calculated
rest_days = this_game.date − same_team.previous_game.date (in days)
Example
Lions play Sun 12/14, next game Thu 12/18 → 4 rest days (short week).
calcPrime-time flagGame ›
Where it's from

Derived from each game's kickoff timestamp.

Sources

SofaScore kickoff datetime → games.date.

How it's calculated
is_primetime = weekday+time ∈ {Thu night, Sun ≈20:20 ET, Mon night}
Example
Kickoff Mon 20:15 ET → is_primetime = true (Monday Night Football).
apiWeatherGame ›
Where it's from

On neither scraper — pulled from a free weather API near kickoff.

Sources

Stadium lat/lon from SofaScore teams + open-meteo forecast.

How it's calculated
open-meteo(venue.lat, venue.lon, game.datetime) → temp, wind, precip · domes flagged "controlled"
Example
Highmark Stadium, 1:00pm → 28°F, wind 18mph, snow → tag "cold / windy".
histTurnover differentialTeam ›
Where it's from

Aggregated from play-by-play, not shown as a stat anywhere.

Sources

nflverse play-by-play (history) + current box scores → team_game_stats.

How it's calculated
turnover_diff = Σ takeaways − Σ giveaways (over the season window)
Example
Lions: 24 takeaways − 14 giveaways = +10.
histPace of playTeam ›
Where it's from

Computed from every offensive snap in play-by-play.

Sources

nflverse play-by-play → team_game_stats.

How it's calculated
sec_per_play = Σ neutral-situation play clock ÷ offensive plays (+ plays/game)
Example
68 plays/game at 26.5 sec/play → fast-paced offense.
histRun / pass tendencyTeam ›
Where it's from

Play-type share, filtered to neutral game script so blowouts don't skew it.

Sources

nflverse play-by-play → team_game_stats.

How it's calculated
pass_rate = pass_attempts ÷ (pass_attempts + rush_attempts) [neutral only]
Example
62% pass on neutral downs → pass-leaning identity.
histRed-zone efficiencyTeam ›
Where it's from

Derived from drives that reach the opponent's 20-yard line.

Sources

nflverse play-by-play → team_game_stats.

How it's calculated
rz_td_pct = red_zone_touchdowns ÷ red_zone_trips
Example
22 TD / 38 trips = 57.9% red-zone TD rate.
histThird-down conversionTeam ›
Where it's from

Counted from every 3rd-down play in play-by-play.

Sources

nflverse play-by-play → team_game_stats.

How it's calculated
third_down_pct = third_downs_converted ÷ third_downs_faced
Example
78 / 174 = 44.8% conversion rate.
calcQB vs defenseMatchup ›
Where it's from

Nobody scrapes this — it's a join of the QB's own game logs against this opponent.

Sources

player_game_stats (nflverse + SofaScore) × opponent team_game_stats / rank.

How it's calculated
avg(yds, TD, INT, rating) of QB's games WHERE opponent = this_team (or vs same-ranked pass Ds)
Example
Goff vs Seahawks, last 4 meetings → 265 yds, 1.8 TD, 0.5 INT, 98.4 rating.
calcWR vs secondaryMatchup ›
Where it's from

Receiver's production sliced by how good the opposing pass defense is.

Sources

player_game_stats × opponent pass-defense rank in team_game_stats.

How it's calculated
avg(rec, yds, TD) of WR's games bucketed by opponent pass-D tier (top-10 / mid / bottom-10)
Example
St. Brown vs top-10 pass Ds → 6 rec, 74 yds; vs bottom-10 → 8 rec, 112 yds.
calcRB vs run defenseMatchup ›
Where it's from

Back's rushing lines matched against the opponent's ability to stop the run.

Sources

player_game_stats × opponent run-defense rank (yards/att allowed) in team_game_stats.

How it's calculated
avg(rush_yds, TD, ypc) of RB's games bucketed by opponent run-D tier
Example
Gibbs vs bottom-10 run Ds → 108 yds, 5.1 ypc, 1.2 TD.
calcOffensive line vs pass rushMatchup ›
Where it's from

Protection vs pressure — one team's sacks-allowed rate against the other's sack rate.

Sources

Both sides from team_game_stats (nflverse pbp): sacks/pressures allowed vs made.

How it's calculated
matchup_index = offense.sack_rate_allowed − defense.sack_rate_generated
Example
OL allows 4.5% sack rate vs a D generating 9.0% → edge: defense (QB under pressure).
calcCoaching tendenciesMatchup ›
Where it's from

Aggregated play-calling patterns per coach/team; some flavor from scraped previews.

Sources

nflverse play-by-play grouped by team/season → team_game_stats.

How it's calculated
profile = { early-down pass rate, PROE, 4th-down go rate, pace } per team
Example
HC goes for it on 4th down 22% (top-5 aggressive), 58% pass on early downs.
